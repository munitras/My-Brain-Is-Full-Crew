## The "Confluence Point" Workflow

1. **Context Set**: User types `@now --switch -to "Research X"`. Chronos logs the start.

2. **Mining**: User triggers Archaeologist. `ralph.sh` mines external URLs/Files into the Scratchpad.

3. **Synthesis**: User triggers Synthesizer. It reads the Scratchpad and drafts a Delegation Map in `01-Projects/Drafts/`.

4. **Morning HUD**: Next day, `ON_START` displays the Chronos report and the new Delegation Map for review.
